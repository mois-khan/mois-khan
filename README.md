### 🚀 Functional Prototypes

#### 🛡️ [CallShield-AI](https://github.com/mois-khan/callshield-ai)
A real-time vishing and social engineering detection pipeline for live telephonic streams. 
* **Architecture:** Built an asynchronous WebSocket pipeline connecting Deepgram (Live STT) with Gemini 1.5 Flash (LLM). 
* **Impact:** Achieves sub-3 second end-to-end detection latency without relying on static ML training datasets, focusing instead on high-speed API orchestration.

#### 📍 [SafeRoute](https://github.com/mois-khan/saferoute-AI)
A multi-criteria pathfinding navigation system prioritizing urban safety over traditional distance metrics.
* **Architecture:** Implements custom heuristic logic weighing geospatial signals like police station proximity, hospital density, and street lighting.
* **Impact:** Designed to significantly improve route safety scores in high-risk urban environments.
